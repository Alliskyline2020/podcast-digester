# 贡献者公约 / Contributor Code of Conduct

## 我们的承诺 / Our Pledge

为了营造一个友好、安全、欢迎的环境，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向身份，我们都承诺致力于让每位成员免受骚扰。

我们以互相尊重、建设性反馈、为整体社区福祉负责的方式开展协作。

## 我们的准则 / Our Standards

正面行为示例：

- 对他人展现同理与善意。
- 尊重不同意见、观点与经验。
- 给予建设性反馈，也优雅地接受反馈。
- 承担责任，向受我们错误影响的人道歉，并从中学习。
- 关注的不仅是个人的，更是整个社区的福祉。

不可接受的行为示例：

- 使用涉及性意味的语言或意象、性关注或挑逗。
- 钓鱼、侮辱或贬损的评论；人身或政治攻击。
- 公开或私下的骚扰。
- 未经明确许可发布他人私人信息（如物理或电子地址）。
- 其他在专业环境下可被合理认定为不当的行为。

## 执行责任 / Enforcement Responsibilities

社区维护者负责澄清和执行本公约的行为准则，并会对其认为不当、威胁、冒犯或有害的任何行为采取适当且公平的纠正措施。

## 适用范围 / Scope

本公约适用于所有社区空间，并在个人在公共空间正式代表社区时同样适用。

## 执行 / Enforcement

如发现不当行为，可通过仓库的 Issue 或邮件私下联系仓库维护者。所有投诉都会被及时、公正地审查和处理。
