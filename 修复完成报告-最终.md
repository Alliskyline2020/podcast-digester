**文件**: `backend/app/main.py:1802-1828`
**影响**: 清理代码，提升性能，改善日志管理

### 3. ✅ 修复任务取消竞态条件
**问题**: 任务取消时不等待旧任务完成，可能导致资源竞争
**修复**:
```python
# 修复前：立即删除任务引用
old_task.cancel()
del self.current_tasks[episode_id]

# 修复后：等待任务真正停止
old_task.cancel()
try:
    await asyncio.wait_for(old_task, timeout=5.0)
except (asyncio.TimeoutError, asyncio.CancelledError):
    logger.warning(f"Old task did not stop cleanly")
del self.current_tasks[episode_id]
```
**文件**: `backend/app/pipeline.py:107-118`
**影响**: 防止资源竞争，提高并发安全性

### 4. ✅ 添加导出速率限制
**问题**: PNG 导出是 CPU 密集型操作，无保护可能导致 DoS
**修复**:
- 添加简单的内存 rate limiter
- 限制：60秒内最多3次导出请求
- 超过限制返回 HTTP 429
**文件**: `backend/app/main.py:43-83, 2132-2138`
**影响**: 防止 DoS 攻击，保护服务器资源
