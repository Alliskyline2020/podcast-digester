## 功能介绍

### 字幕同步与映射功能概述

字幕同步与映射功能将零散的字幕片段（segments）智能合并成更易读的段落，并保留原始映射关系，提供更好的阅读体验。

#### 主要特性

1. **自动定位**: 播放器跳转后，字幕列表自动滚动到当前播放位置
2. **映射可视化**: 显示每个段落包含的原始字幕数量和索引
3. **智能分段**: 根据语义和长度自动合并字幕
4. **降级兼容**: 旧数据自动使用前端分段逻辑

## 使用指南

### 1. 字幕自动滚动

#### 功能说明

当您在播放器中跳转到不同时间点时，字幕列表会自动滚动到对应位置，无需手动查找。

#### 使用方法

1. 打开任意节目的播放器页面
2. 切换到"转录字幕"标签页
3. 点击播放器进度条跳转到任意位置
4. 字幕列表自动滚动到当前播放位置对应的段落
5. 当前段落会高亮显示，方便识别

#### 视觉提示

- **高亮段落**: 当前播放位置对应的段落会有不同的背景色
- **平滑滚动**: 列表平滑滚动到目标位置
- **居中显示**: 当前段落尽量居中显示在可视区域

---

### 2. 查看字幕映射

#### 功能说明

每个字幕段落都会显示其包含的原始字幕数量和索引范围，帮助您了解段落的组成。

#### 使用方法

1. 在播放器页面的"转录字幕"标签页
2. 每个段落下方显示"X 段原始字幕 #1 - #5"
3. 点击该信息可展开查看详细的原始字幕列表
4. 再次点击可收起详情

#### 映射信息说明

```
3 段原始字幕 #1 - #3
```

- **3 段原始字幕**: 该段落由 3 个原始字幕片段组成
- **#1 - #3**: 原始字幕的索引范围（从第1段到第3段）

#### 展开详情

点击后显示：
- 每个原始字幕的索引（#1, #2, #3）
- 每个原始字幕的唯一 ID（seg_001, seg_002, seg_003）

---

### 3. 字幕显示模式

#### 支持的显示模式

1. **原文模式**: 仅显示原始语言字幕
2. **翻译模式**: 仅显示翻译后的字幕
3. **双语模式**: 同时显示原文和翻译

#### 切换方法

在字幕列表顶部，点击"显示"下拉菜单，选择您想要的模式。

#### 推荐使用场景

- **原文模式**: 学习原语言、查看原始表达
- **翻译模式**: 快速理解内容
- **双语模式**: 对比学习、精确理解

## 高级功能

### 批量字幕同步（管理员）

#### 功能说明

对于旧数据（没有段落映射的节目），管理员可以批量生成字幕分段映射。

#### 使用方法

**使用 API:**

```bash
curl -X POST http://your-domain.com/api/admin/batch-sync-subtitles \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -d '{
    "episode_ids": ["ep_001", "ep_002", "ep_003"]
  }'
```

**响应示例:**

```json
{
  "total": 3,
  "successful": ["ep_001", "ep_002"],
  "failed": [
    {
      "episode_id": "ep_003",
      "error": "字幕文件不存在"
    }
  ],
  "duration_ms": 123
}
```

#### 使用场景

- 新功能部署后批量处理现有数据
- 字幕分段规则调整后重新生成
- 修复之前同步失败的节目

## 智能分段规则

### 分段逻辑

字幕段落根据以下规则自动生成：

1. **最大长度**: 每段最多 120 个字符
2. **最小长度**: 每段最少 40 个字符
3. **时间间隔**: 如果两个字幕间隔超过 2 秒，强制分段
4. **语义完整**: 优先保证段落的语义完整性

### 分段示例

```
原始字幕:
[0:00-0:05] 第一句话
[0:05-0:10] 第二句话
[0:10-0:15] 第三句话

合并为段落:
"第一句话 第二句话 第三句话"
（包含 3 段原始字幕）
```

## 常见问题

### Q1: 为什么有些节目没有"X 段原始字幕"的标识？

**原因**: 这些节目是旧数据，没有段落映射信息。

**解决**: 系统会自动使用前端分段逻辑生成段落，不影响使用。如需完整映射功能，请联系管理员批量同步。

### Q2: 字幕自动滚动不工作？

**原因**: 可能是浏览器兼容性问题或数据加载延迟。

**解决**: 
1. 刷新页面重试
2. 检查浏览器控制台是否有错误
3. 使用最新版本的 Chrome、Firefox 或 Safari

### Q3: 点击"X 段原始字幕"没有反应？

**原因**: JavaScript 加载失败或组件异常。

**解决**: 
1. 刷新页面
2. 清除浏览器缓存
3. 检查网络连接

### Q4: 为什么段落有时候很长？

**原因**: 字幕分段规则优先保证语义完整，某些情况下会合并多个短句。

**建议**: 这是正常行为，有助于保持上下文完整性。

### Q5: 如何报告问题或提出建议？

**途径**: 
1. 在项目 GitHub 页面提交 Issue
2. 联系项目维护者
3. 在用户社区反馈

## 键盘快捷键

### 播放器快捷键

- **空格**: 播放/暂停
- **左/右箭头**: 快退/快进 5 秒
- **上/下箭头**: 音量调节
- **J**: 快退 10 秒
- **K**: 播放/暂停
- **L**: 快进 10 秒

### 字幕列表快捷键

- **点击段落**: 跳转到对应时间
- **点击映射信息**: 展开/收起原始字幕详情
