## 技术架构改进

### Pipeline 阶段（8阶段）
```
1. 下载 (20%)      → 音频文件下载
2. 转录 (20%)      → ASR 语音识别 / YouTube 字幕下载
3. 分章 (10%)      → 拆分章节大纲
4. 摘要 (17%)      → 生成章节摘要
5. 翻译 (18%)      → 翻译为中文（仅非中文内容）
6. 亮点 (18%)      → 提取亮点金句
7. 产品洞察 (15%)  → 提取产品和技术洞察 ⭐ 新增
8. 持久化 (0%)     → 保存结果到文件系统和数据库
```

### 双语字幕数据流
```
YouTube 下载 → VTT 文件解析 → 双语合并
    ↓              ↓              ↓
zh-Hans,zh,en  parse_vtt()  _merge_bilingual_transcripts()
    ↓              ↓              ↓
  两个文件      中/英分别      中文 text_original
                  解析            英文 text_translated
```

### 数据持久化策略
```
文件系统              数据库
─────────────────────────────────
transcript.json      → transcript (JSON)
outline.json         → outline (JSON)
summaries.json       → summaries (JSON)
highlight.json       → highlight (JSON)
product_insights.json → product_insights (JSON) ⭐ 新增
```
