# 字幕显示优化方案

## 🎯 问题分析

### 当前问题
1. **切分太细** - 120字符限制导致段落被切碎
2. **无文本清洗** - 包含语气词、特殊符号
3. **缺少句子映射** - 无法追溯到句子级别的原始字幕

---

## 💡 优化方案

### 1. 按内容段落切分（语义分句）

**当前策略（问题）：**
```python
SubtitleSegmenter(max_chars=120, min_chars=40)
# 结果：段落被切碎，阅读体验差
```

**改进策略（推荐）：**
```python
SubtitleSegmenter(max_chars=300, min_chars=80, use_semantic=True)
# 结果：按语义段落切分，更符合自然阅读
```

**语义切分规则：**
- 检测句子结束符（。！？. ?）
- 检测段落结束符（连续换行、说话人变化）
- 优先保持语义完整性

### 2. 文本清洗

**需要清洗的内容：**
```python
# 语气词
"嗯", "啊", "哦", "呃", "那个", "然后"

# 特殊符号
<b>, </b>, <i>, </i>, [音乐], [笑声]

# 冗余空格
"  多个空格  "
```

**清洗函数：**
```python
def clean_subtitle_text(text: str) -> str:
    # 移除HTML标签
    text = re.sub(r'<[^>]+>', '', text)

    # 移除语气词
    filler_words = ['嗯', '啊', '哦', '呃', '那个', '然后']
    for word in filler_words:
        text = text.replace(word, '')

    # 清理空格
    text = re.sub(r'\s+', ' ', text).strip()

    return text
```

### 3. 句子级别映射

**新增数据结构：**
```json
{
  "id": 0,
  "start_ms": 12742,
  "end_ms": 32628,
  "text_original": "大家好我是小军...",
  "text_clean": "大家好我是小军...",
  "sentences": [
    {
      "id": 0,
      "text": "大家好我是小军",
      "start_ms": 12742,
      "end_ms": 15000,
      "segment_id": 0,
      "segment_index": 0
    },
    {
      "id": 1,
      "text": "本集节目我们来到了美国纽约",
      "start_ms": 15000,
      "end_ms": 18000,
      "segment_id": 1,
      "segment_index": 1
    }
  ],
  "segment_indices": [0,1,2,3,4,5,6,7,8],
  "segment_ids": [0,1,2,3,4,5,6,7,8]
}
```

**为什么需要句子映射？**
- ✅ 金句提取：追溯到具体句子
- ✅ 时间戳：精确到句子级别
- ✅ 引用标注：显示原文位置
