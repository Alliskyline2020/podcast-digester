## ⚠️ 已知限制

### 数据结构问题
API返回的数据有双重嵌套：
```json
{
  "episode": {
    "episode": {
      "id": "ep_1781109978390",
      ...
    }
  }
}
```

**原因**: EpisodeResponse模型可能是旧版本，需要调整
**影响**: 前端可能需要调整数据访问路径（`episode.episode.episode`）

### 临时解决方案
前端可以临时使用：
```javascript
const episode = data.episode?.episode || data.episode
```
